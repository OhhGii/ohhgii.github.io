01

OTA 行业情报雷达

面向酒店收益经理的行业情报自动化 Skill，替代每日人工多信源浏览。

Harness 工程 LLM 精筛 双视角解析 自动化闭环 Skill 设计

针对酒店收益经理定价决策时缺失的对手盘反应能力，打造行业雷达 Skill 及推送机制。收益经理日常需跟踪 21 个信源的行业动态，但传统浏览只能获取表层信息，缺少"对手会怎么反应"的推演层。

核心方法论是 Harness 工程——Agent = Model（随机智能）+ Harness（确定性基础设施）。在开发态用 GSD 结构化工作流驱动 AI Coding Agent，约 110 分钟从零交付完整系统；运行态用 600 行规则文档 + 程序化质检约束 AI 每日输出。

全链路自动化：21 信源抓取 → 关键词白名单粗筛 → LLM 精筛 → 五维度打分制分级 → 双视角深度解析（内部酒店策略 + 外部 OTA 对手盘推演）→ 跨信号全局叙事 → 飞书 Wiki 五章节简报推送。收益经理 3 分钟完成每日情报消费。

多平台爬虫 Skill

支持微信公众号、小红书、微博、抖音等强反爬平台的通用数据采集工具。

Playwright CDP RPA 反爬 Python

双引擎架构设计：普通网页使用 FastEngine（HTTP 直接请求 + Playwright 兜底），强反爬平台使用 RPAEngine（真实浏览器 RPA + CDP 连接 + 反爬脚本注入）。通过平台规则引擎自动识别目标并推荐采集模式。

登录态自动复用，首次手动登录后持久化保存；支持列表页批量提取与详情页自动跳转；导出 CSV / JSON 格式。覆盖微信公众号、小红书、微博、抖音等主流平台。

AI 叙事 拍照识别 降级链路 混合架构

独立负责的 AI 健康饮食管理产品，聚焦年轻群体减脂/增肌场景。采用"确定性计算 + AI 叙事"混合生成架构，将 BMR / TDEE 等核心指标交由传统代码计算，AI 仅负责个性化叙事，从底层机制杜绝大模型幻觉风险。
